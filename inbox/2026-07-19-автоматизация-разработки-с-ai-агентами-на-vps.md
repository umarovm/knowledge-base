# Автоматизация разработки с AI-агентами на VPS

Идея полностью изолировать environment для кодинга на отдельный VPS-сервер. Установить OpenClaw или Hermes agent (не уверен, какой лучше). Подключить одну папку из Knowledge Base со списком тасков по проектам для кодинга. AI-агент будет просыпаться в 9 часов утра, смотреть таски, кодить, дебажить, фиксить баги и пушить в GitHub репозиторий проекта. Затем обновлять таскборд о выполненной работе и пушить обновления в Knowledge Base, откуда они подтягиваются ко мне. Нужно настроить GitHub Actions для проверки pull requests перед мержем с main бренчем. Для работы в пределах лимитов использовать OpenRouter/OmniRoute и API от ChatGPT. ChatGPT 5.6 Sol будет менеджером, создав структуру с исполнителями и теми, кто чистит баги, как маленькую компанию агентов. DeepSeek и Flash будут работать как исполнители, дебаггеры, чистильщики кода, а дорогие модели оркестрировать. Создай некий loop engierring/workflow для этих агентов.
