# AionUi — рабочий стол для ИИ-агентов

https://github.com/iOfficeAI/AionUi

Пост из tg-канала (t.me/c/1975179926/331): опенсорсный кроссплатформенный GUI для CLI ИИ-агентов (Windows/macOS/Linux). Сканирует установленные CLI-агенты (Claude Code, Codex, Cursor Agent и др.) и даёт единый интерфейс управления; есть встроенный агент без установки CLI (Gemini/OpenAI/Anthropic/DeepSeek/Ollama по ключу); Team Mode — команда агентов с лидером, делит задачи и делегирует через ACP-протокол на канбан-доске; просмотрщик 10+ форматов (код, HTML, Markdown, PDF, Word, Excel, PPTX) прямо в приложении; расписание задач на естественном языке (cron 24/7); встроенные утилиты OfficeCLI — презентации с анимацией (Morph PPT), Word-документы и шаблоны договоров, Excel с формулами и графиками; веб-интерфейс и Telegram-бот для управления с телефона. Данные хранятся локально в SQLite.

Нужно проверить и решить, стоит ли использовать для продуктивности — актуально в контексте текущей работы с Claude Code над LLMWikiBot и трейдинг-ботами.
