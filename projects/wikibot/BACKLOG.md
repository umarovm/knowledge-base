# BACKLOG — LLMWikiBot

## Открыто

| Добавлено | Пункт |

| 2026-07-16 | Опция «сохранить как есть» без обработки Haiku: после любого входящего сообщения бот спрашивает двумя кнопками — [Сохранить как есть] / [Haiku]. Аудио → всегда транскрибируется в текст, дальше по выбору кнопки: сохранить транскрипт как есть или прогнать через Haiku. Текст/текст+PDF → сохраняется как есть по умолчанию (PDF прикладывается), либо обрабатывается Haiku по выбору. |
